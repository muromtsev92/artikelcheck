/nouns
GET /nouns - возвращает страницу со всеми объектами
GET nouns/game - возвращает страницу со случайными словами
GET nouns/add - возвращает страницу с формой ввода слова
GET nouns/get-rnd - возвращает в виде json 3 случайных слова
GET nouns/delete/{id} - удаляет по id

POST nouns/add - добавить существительное в базу
POST nouns/update/{id} - обновить существительное
